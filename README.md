Nigerian Foods App

A simple web app where you can discover Nigerian dishes from all parts of the country. You can browse, search, and learn about different meals easily.

Live site: https://nigerian-foods-u37g.vercel.app/

What the App Does
- Shows a list of popular Nigerian foods.
- Uses an API to fetch foods
- Allows users to search for foods by name or description
- Allows users to filter foods by:
	- Category
	- Region
	- Vegetarian
	- Spicy
- Displays details about each meal when you click on the view details button.
- Allows you to add or remove foods from favorites
- Works on both desktop and mobile.

How to Use
- Open the website.
- Browse meals or use the search box to find one.
- Click on a meal to see more details.
- Use the category filter to see specific types of food.
